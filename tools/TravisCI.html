    Travis CI¶

    tl;dr

    Use the Xenial environment, enable Docker and you're all set.

    Travis CI's Xenial environment includes everything batect requires, so all that needs to be done to use batect with Travis CI is to enable the Docker service.

    Adding the following to your .travis.yml file selects the Xenial environment and enables Docker:

    dist: xenial

services:
  - docker

    You can see a full example of using batect with Travis CI in the Java sample project.
